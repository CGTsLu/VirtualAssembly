# 砌体施工流程VR仿真

#### 项目简介
在虚拟现实环境下，实现建筑行业中砌体工艺流程的仿真操作。用户佩戴VR设备可以进行施工仿真的操作交互，让用户在沉浸式环境中积累砌体施工经验。
